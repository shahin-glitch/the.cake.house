# 🎂 the.cake.house Website

## 📌 Project Overview

This is a simple and responsive **the.cake.house Website** where users can explore different types of cakes and place orders easily through WhatsApp.

---

## 🚀 Features

* 🎨 Beautiful and clean UI
* 🍰 Multiple cake categories (Birthday, Wedding, Bento, etc.)
* 📱 WhatsApp order integration
* 🖼️ Cake images with details
* ⚡ Fast and responsive design

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
